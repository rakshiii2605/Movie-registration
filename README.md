# 🎬 Movie Ticket Booking System

This project is a web-based Movie Ticket Booking System developed using Flask and MongoDB. It allows users to register, log in, browse available movies with posters, select seats, and book tickets easily.

---

## 🚀 Features

- User Registration and Login System  
- Browse Movies with Posters  
- Seat Selection  
- Ticket Booking  
- Booking History  
- MongoDB Atlas Cloud Database Integration  
- Simple and Clean UI  

---

## 🛠️ Tech Stack

- **Backend:** Flask (Python)  
- **Database:** MongoDB Atlas  
- **Frontend:** HTML, CSS  
- **Tools:** PyMongo
